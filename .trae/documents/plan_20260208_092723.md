Uygulamanın web sürümünü localhost üzerinde başlatmak için aşağıdaki adımları izleyeceğim:

1.  **Mevcut Sunucuyu Durdurma:** Şu anda arka planda çalışan web sunucusu işlemini sonlandırarak temiz bir başlangıç yapacağım.
2.  **Sunucuyu Başlatma:** `flutter run -d web-server --web-port=8080 --web-hostname=0.0.0.0` komutu ile uygulamayı web sunucusu modunda yeniden başlatacağım.
3.  **Önizleme:** Sunucu hazır olduğunda size önizleme bağlantısını sunacağım.
